
# ⚡ Pokedex Lite

A beautiful, responsive web application that allows users to browse, search, filter, and favorite Pokémon from the PokéAPI.

## ✨ Features

### ✅ Mandatory Features (All Completed)

- **Data Fetching** - Fetches Pokémon data from the official PokéAPI
- **Responsive Grid Layout** - Displays Pokémon in a responsive grid that works on mobile, tablet, and desktop
- **Search Functionality** - Real-time search by Pokémon name
- **Type Filtering** - Filter Pokémon by their types (Fire, Water, Grass, etc.)
- **Pagination** - Navigate through pages with Previous/Next buttons
- **Favorites System** - Mark Pokémon as favorites with persistent storage in localStorage
- **Detail Modal** - View detailed information including stats, abilities, height, and weight
- **Loading States** - Shows loading spinner while fetching data
- **Error Handling** - Displays user-friendly error messages if API fails

### 🎨 Bonus Features

- **Smooth Animations** - Hover effects, card transitions, and modal animations
- **Type-Based Colors** - Each Pokémon type has its own unique badge color
- **Responsive Design** - Fully responsive across all device sizes

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Modern styling with CSS Grid, Flexbox, and gradients
- **Vanilla JavaScript** - No frameworks, pure JavaScript for better performance
- **PokéAPI** - RESTful API for Pokémon data
- **LocalStorage API** - For persisting favorite Pokémon

### Why Vanilla JavaScript?

Instead of using React or other frameworks, I chose vanilla JavaScript for:
- **Zero dependencies** - No npm install or build steps required
- **Faster load times** - No framework overhead
- **Better learning value** - Demonstrates core web development skills
- **Easy deployment** - Single HTML file works anywhere

## 📋 Prerequisites

No prerequisites! This is a pure HTML/CSS/JS application that runs in any modern web browser.
